This project is a very basic todo -list app created using reactjs forthe front endd and firebase forthe back end .
Adding list item to todo and updating database are the 2 functinalities.
Do install the node modules
 $ npm install
firebase is used forthe  back end of this project
  $npm install -g firebase-tools
  $firebase login
  $npm i firebase   
Material ui is also used inthe styling of this project
  $npm install @material-ui/core or $npm install @mui/material @emotion/react @emotion/styled
